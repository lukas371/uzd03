HTML Paveikslėliai

HTML kalboje paveikslėliai yra įterpiami per žymą ‹img›. ‹img› žyma yra VOID elementas, ji taigi, ji neturi uždarymo žymos ir negali laikyti turinio. Paveikslėlio dydis yra keičiamas CSS taisyklių pagalba. ‹img› taip pat turi width ir height parametrus, kurie nurodo paveikslėlio dydį pikseliais. Visgi, geriausia praktika - šių atributų nenaudoti, o paveikslėlio dydį keisti aprašant CSS stiliaus taisykles.
